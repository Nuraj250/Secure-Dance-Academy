# Assumptions

- Users will access the system using modern browsers.
- The organization will control user onboarding and role assignment.
- Sensitive records require authorization before disclosure.
- The approved Task 01 platform choices remain fixed.
- The academy operates on a single-site baseline for the current requirements phase.
- Parents and coaches are linked to the artists they are permitted to view or manage.

## Assumption Notes

- If any assumption changes, the requirements baseline must be updated before design
  work continues.
